# react-storybook-msw-vitest

## storybookとmswとvitestを繋ぎモックAPIを用いたテスト  
※フォーム送信コンポーネントのみ  

## ボタンコンポーネントリスト  
・ActionButton	Loading→Success→Errorの状態遷移ボタン  
・ToastButton	ボタン押下でトースト通知(モックAPI)  
・FetchDisplayButton	APIデータを取得して画面に表示  
・ModalActionButton	モーダルを開き、モーダル内でAPI実行  
・ToggleButton	ON/OFFの状態切替ボタン(API通信あり)  

## フォーム送信コンポーネントリスト  
・FormSelectSubmit  セレクトボックス+フォーム送信
